# Library Book Manager CLI

A command line application to manage library books built with Node.js.

## Features
- Add a new book
- View all books
- Delete a book by ID
- Search books by title

## Tech Stack
- Node.js
- fs module

## How to Run
node index.js add "Title" "Author" Year
node index.js view
node index.js delete <id>
node index.js search "keyword"
